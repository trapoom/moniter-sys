moniter

Secure Debian and Ubuntu Package Repository

Quick Installation

Step 1 — Configure Repository
curl -fsSL https://trapoom.github.io/debian-apt-push/setup.sh | sudo bash
Step 2 — Install Package
sudo apt install moniter
Features
  • Secure GPG signature verification
  • Automatic repository configuration
  • Compatible with Debian and Ubuntu
  • APT signed-by keyring support
  • Safe to run multiple times
